# 진단 보고서 템플릿 (KISA 2021 전수 점검)

진단 결과는 반드시 아래 구조로 출력한다. **설계 20개 + 구현 49개 전 항목**을
결과표에 빠짐없이 채워야 하며, 어떤 항목도 생략할 수 없다.
판정은 `PASS` / `VULNERABLE` / `NEEDS_REVIEW` / `N/A`(사유 필수) 중 하나.

---

## 1. Executive Summary
- 대상/범위, 진단일, 기술스택
- 종합 통계: 설계 20개 중 PASS/VULN/REVIEW/NA 개수, 구현 49개 중 PASS/VULN/REVIEW/NA 개수
- 핵심 위험 Top N

## 4. 상세 발견사항 (VULNERABLE / NEEDS_REVIEW 전부)
각 발견마다:
- 항목 번호/이름, 위치(파일:라인)
- 설명 / 공격 시나리오
- 영향도(심각도)
- 수정 가이드 + 예시 코드(전/후)

## 5. 수정 우선순위 (Patch Plan)
P0/P1/P2로 분류한 조치 목록과 예상 난이도.
